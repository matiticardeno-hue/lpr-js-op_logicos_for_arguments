🧩 Descripción

Este trabajo práctico profundiza los fundamentos de programación en JavaScript, incorporando estructuras de decisión, validaciones lógicas y resolución de problemas algorítmicos.

A través de ejercicios prácticos, se desarrollan habilidades relacionadas con:

lógica computacional
toma de decisiones
validación de datos
análisis matemático
uso de estructuras condicionales
🎯 Objetivos de Aprendizaje

Al completar este práctico, el estudiante será capaz de:

Implementar funciones utilizando estructuras condicionales
Resolver problemas mediante lógica booleana
Comparar valores numéricos y textuales
Aplicar operaciones matemáticas y validaciones
Utilizar algoritmos simples para resolver problemas complejos
🛠️ Capacidades y Habilidades Adquiridas
🔹 1. Uso de estructuras condicionales

Durante el práctico se trabaja con:

if
else
else if

Estas estructuras permiten que el programa tome decisiones según distintas condiciones.

Habilidades desarrolladas:
Evaluación lógica
Toma de decisiones en programación
Construcción de algoritmos condicionales
🔹 2. Comparación de valores

Se utilizan operadores de comparación como:

>
<
>=
<=
===
!==
Aplicaciones:
Determinar números mayores
Validar rangos
Comparar valores exactos
🔹 3. Lógica booleana

El práctico incorpora expresiones que devuelven:

true
false

y operadores lógicos como:

&& (AND)
|| (OR)
Habilidades desarrolladas:
Resolución lógica
Interpretación de condiciones múltiples
Validación de datos
🔹 4. Funciones y modularización

Cada problema se resuelve mediante funciones independientes.

Esto permite:
Reutilizar código
Organizar soluciones
Separar responsabilidades
Competencias adquiridas:
Diseño modular
Programación estructurada
Lectura y mantenimiento de código
🔹 5. Validación matemática

Los ejercicios trabajan conceptos matemáticos aplicados a programación:

Divisibilidad

Operaciones con módulo (%)
